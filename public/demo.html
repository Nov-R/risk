🔍 风险管理 API

GET /api/risks

获取所有风险列表

POST /api/risks

创建新的风险记录

GET /api/risks/{id}

获取指定风险详情

PUT /api/risks/{id}

更新风险信息

DELETE /api/risks/{id}

删除风险记录

💬 反馈管理 API

POST /api/feedbacks

创建反馈记录

GET /api/feedbacks/{id}

获取反馈详情

GET /api/risks/{riskId}/feedbacks

获取风险相关反馈

GET /api/feedbacks/status/{status}

按状态查询反馈

🔄 节点审核 API

POST /api/nodes

创建审核节点

POST /api/nodes/{id}/approve

审批通过节点

POST /api/nodes/{id}/reject

拒绝审核节点

GET /api/nodes/pending/{type}

获取待处理节点

2. 启动服务: docker-compose up -d
